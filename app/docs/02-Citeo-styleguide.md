## STYLES / COULEURS

La palette de couleur choisie est volontairement restreinte afin de calmer le foisonnement d’éléments Ui qui peuplent l’interface.
Elle s’articule autour de 3 ensembles : la SuperColor, les Couleurs Secondaires et les Couleurs Invariantes.

### SuperColor

Chaque Public sera caractérisé par sa SuperColor.
Dans cette version du Playbook, la SuperColor utilisée pour exemple est celle de l’Espace Entreprise.
